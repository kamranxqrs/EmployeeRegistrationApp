# Employee Registration Application

A web-based Employee Registration and Login system built using Spring, Hibernate, JSP, and MySQL.

## Features
- Employee registration with personal and login details  
- Secure login with session management  
- Welcome page displaying all registered employees  
- Logout functionality with session invalidation  

## Technologies Used
- Java 11  
- Spring 
- Hibernate ORM  
- JSP and JSTL  
- MySQL  
- Apache Tomcat  
- Maven  

## Project Structure
- Controller layer for handling HTTP requests  
- Service layer for business logic  
- DAO layer for database operations  
- JSP for the view layer  
- Hibernate for ORM and database mapping  

## How to Run
1. Clone the repository  
2. Import the project into Eclipse as a Maven project  
3. Configure the MySQL database and update `hibernate.cfg.xml`  
4. Deploy the project on Apache Tomcat  
5. Open the application in the browser at:  
http://localhost:8080/EmployeeRegistrationApp/login
